# StarBeat~
> 星之鼓动~

## 介绍

StarBeat系统是一个基于Nuxt+Typecho的轻量级博客系统

### 快速开始
```bash
git clone https://github.com/ShuShuicu/StarBeat.git
```

#### 安装依赖
```bash
cd StarBeat
npm i @nuxtjs/seo
npm i prismjs
npm i nprogress
npm i -D vuetify vite-plugin-vuetify
npm i @mdi/font
```

#### 运行&构建
```bash
npm run dev
npm run build
```
